## Project Structure

```bash
webproj/
├─ README.md
├─ public/
│  ├─ main.html
│  ├─ css/
│  │  └─ styles.css
│  ├─ components/      
│  └─ pages/          
└─ assets/
   └─ img/
   └─ fonts/             
```

- `public/` : 정적 루트(브라우저가 여는 폴더)
- `main.html` : 메인 페이지
- `css/` : css 파일 폴더
- `styles.css`: 공통 스타일
- `components/`: 공통 파셜 폴더 (ex.헤더, 푸터 필요시 활용)
- `pages/` : 서브페이지 폴더
- `assets/ ` : 이미지, 아이콘, 폰트 등 에셋 파일 폴더
